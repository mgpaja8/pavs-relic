# Pavs Relic

#### Prerequisites

1. Install Docker: https://docs.docker.com/desktop/install/mac-install/

#### Run project

1. Run `docker-compose up`

#### Accessing app

1. Front end is running on port 3000
2. API is running on port 3001
3. Postgres db is running on port 15432
